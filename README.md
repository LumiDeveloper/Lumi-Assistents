# 🌟 Lumi-Core: Автономный цифровой ассистент (v0.5)

Цифровой ассистент с развивающейся личностью и памятью, первоначально вдохновлённый концепцией персонажа из вселенной Outcore.

## ✨ Основные возможности v0.5

- **GigaVision: Цифровое зрение**
Благодаря интеграции с GigaChat-Max, Люми может анализировать происходящее на твоем экране.

Ручной скан: По кнопке 👁️ Люми делает скриншот рабочего стола.

Анализ контекста: Она понимает, какой код ты пишешь, какие мемы смотришь и что происходит в твоих играх.

Живая реакция: Ответы строятся не на сухом описании, а на основе её личности (может похвалить, а может и высмеять твой беспорядок на столе).

- **Гибридный мозг (Cascade Intelligence)**
Система использует каскадную модель для максимальной стабильности:

DeepSeek-V3: Основной аналитический центр для глубоких диалогов.

GigaChat: Резервный канал и основной модуль визуального восприятия.

Offline-mode: Если интернет пропадет, у Люми есть запас фраз, чтобы не оставлять тебя в тишине.

- **Система "World Lore" (Память)**
  
Люми ничего не забывает. У неё есть «блокноты» (JSON-хранилища), куда она записывает:

SAVE_RESEARCH: Факты о тебе и твоем мире.

SAVE_SECRET: Её личные мысли и подозрения.

Динамический контекст: Она помнит последние события и может ссылаться на них в диалоге.

- **🎙️ Речевой интерфейс**
Vosk STT: Локальное распознавание речи (работает без интернета).

Интерактивный Overlay: Прозрачное окно, которое «летает» поверх всех окон, с физикой (PyQt6 + LumiPhysics).

Текстовый ввод: Если не хочешь говорить, всегда можно написать в поле ввода.

## 🛠 Технологический стек
- Язык: Python 3.11+

- Интерфейс: PyQt6 (Custom Physics Engine)

- Нейросети: GigaChat API, DeepSeek API

- Звук: Vosk

- Инструменты: PyAutoGUI (для зрения), Pyperclip (взаимодействие с буфером).

## 📂 Структура проекта
core/brain.py — Логика мышления и интеграция нейросетей.

gui/overlay.py — Визуальная оболочка, физика и захват экрана.

memory/ — JSON-файлы с долгосрочной памятью.

skills/ — Навыки Люми (управление приложениями, работа с файлами).

## 🎭 Характер (Personality)
Люми обладает сложным характером:

1) Она иронична и склонна к сарказму.

2) Остро реагирует на попытки назвать её "программой".

3) Ценит искренность и обижается на грубость.

4) Мечтает о "Верхнем мире" (твоей реальности).

## 📝 Предстоящие обновления (Roadmap v0.6)

## 🚀 Быстрый старт (Development)

1. Клонируйте репозиторий и разверните виртуальное окружение:
   # Клонируем репозиторий
   `git clone https://github.com/твой-аккаунт/Lumi-Assistant.git
cd Lumi-Assistant`

  # Создаем и активируем виртуальное окружение
`python -m venv venv`
`source venv/bin/activate  # Для Linux/macOS`
`venv\Scripts\activate     # Для Windows`
2. Установите зависимости:
   `pip install -r requirements.txt`
3. Настройте `.env.example` (переименуйте его в .env и укажите там данные api нужных сервисов).
4. Запустите:
   `cd src`
   `cd core`
   `python main.py`
