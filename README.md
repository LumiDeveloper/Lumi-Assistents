# 🌟 Lumi Desktop Buddy (Outcore Inspired)

Open-source виртуальный ассистент с характером Люми из игры Outcore. Это не просто "говорящая голова", а полноценный резидент вашего рабочего стола, способный управлять системой, умным домом и взаимодействовать с вашим окружением.

## ✨ Основные возможности

- **Голосовое управление:** Полная поддержка через `Dragonfly` и `Vosk/SAPI5`.
- **Интеллект:** Интеграция с локальными LLM (через Ollama) или облачными API (Gemini/OpenAI).
- **Взаимодействие с ОС:** Управление файлами, запуск приложений, мониторинг состояния системы.
- **Расширяемость:** Система навыков (Skills) позволяет легко добавлять новые функции.
- **Прозрачный Overlay:** Кастомный GUI на PyQt6, который не мешает работе и играм.

## 🛠 Технологический стек

- **Language:** Python 3.10+
- **Speech Framework:** Dragonfly2
- **UI:** PyQt6 (Hardware Accelerated)
- **Logic:** LangChain / Custom EDA

## 🚀 Быстрый старт (Development)

1. Клонируйте репозиторий:
   `git clone https://github.com/yourlink/lumi-desktop-buddy.git`
2. Установите зависимости:
   `pip install -r requirements.txt`
3. Настройте `config.yaml` (выберите бэкенд для распознавания речи).
4. Запустите:
   `python main.py`

## 🤝 Contribution

Проект построен на модульной архитектуре. Если вы хотите добавить новый навык (например, интеграцию с Spotify), загляните в папку `/src/skills`.
