# 🌟 Lumi-Core: Автономный цифровой ассистент (v0.5)

Цифровой ассистент с развивающейся личностью и памятью, первоначально вдохновлённый концепцией персонажа из вселенной Outcore.

## ✨ Основные возможности v0.5

- **Безопасность:** Полный переход на `.env` файлы.
- **Каскадная система API:** Если DeepSeek не отвечает программа самостоятельно переключается на Gigachat (русский аналог).
- **Интерфейс:** Исправлены ошибки отображения облачка.
- **Офлайн-режим:** Люми не молчит при отсутствии интернета, она общается локальными репликами.

## 🛠 Технологический стек

- **Language:** Python 3.10+
- **UI:** PyQt6 (Hardware Accelerated)

## 🚀 Быстрый старт (Development)

1. Клонируйте репозиторий и разверните виртуальное окружение:
   `git clone https://github.com/LumiDeveloper/Lumi-Assistens.git`
   `cd Lumi-Assistens`
   `python -m venv .venv`
   `.venv\Scripts\activate`
2. Установите зависимости:
   `pip install -r requirements.txt`
3. Настройте `.env.example` (переименуйте его в .env и укажите там данные api нужных сервисов).
4. Запустите:
   `cd src`
   `cd core`
   `python main.py`
